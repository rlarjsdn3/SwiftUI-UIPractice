# 스위프트UI 미니 프로젝트
* `SwiftUI`를 활용해 코딩한 학습 목적의 미니(단기) 프로젝트를 모아 놓은 저장소입니다.

## 알아두세요!
* 미니 프로젝트의 목표는 실제 앱의 구동이 아닌 `학습용`입니다. 일부 기능이 완성되지 않거나 비어있을 수 있습니다.
